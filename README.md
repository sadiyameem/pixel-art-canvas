# Pixel Art Canvas

# Description
- This is a simple drawing app that allows users to create their own grid and draw by clicking and dragging over the squares.
- Users can choose colors, switch betwwen drawing and erasing, and resetting the grid.

# Features
- Create grid
- Click and draw to draw
- Color picker
- Erase button
- Clear button

# Tools
- HTML
- CSS
- JavaScript

# Challenge
- Making the canva appear and drawing function work

# Learned
- The importance of ordering code for example the event listeners
